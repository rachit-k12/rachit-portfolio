## Design Philosophy (Black/White, Minimal, Modern)

**Colors:**
- Black background (#000000 or very dark)
- White text (#FFFFFF)
- Gray for secondary text (rgba(255,255,255,0.6))
- NO other colors (no gradients, no blue, no purple)

**Typography:**
- Clean sans-serif (Inter, Geist, or system fonts)
- Large headings (60-80px for hero)
- Readable body text (16-18px)
- Proper line height (1.6-1.8)

**Layout:**
- Lots of whitespace
- Generous padding
- Max-width containers (1200px)
- Grid-based layouts

**Animations:**
- Minimal, subtle
- Fade in on scroll
- Smooth transitions
- Lenis smooth scroll
- NO crazy effects, NO glassmorphism, NO neomorphism

**Modern 2025-2026 Design Trends:**
- Subtle borders (1px, rgba(255,255,255,0.1))
- Border-radius: 12-16px for cards, 8px for buttons
- Hover states: subtle opacity changes or underlines
- No drop shadows (or very subtle: 0 2px 8px rgba(0,0,0,0.1))
- Text-based design, let content speak
- Gallery-driven layouts
